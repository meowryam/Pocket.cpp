# Pocket++

> *"Doraemon, get me the—" wait, let me check the inventory first.*

A console-based C++ app inspired by Doraemon's 4D Pocket, built to actually manage the chaos of hundreds of gadgets: what's available, who borrowed what, and what to pack for the next mission. (Curious? There's a full software handbook in the docs folder.)

## Status: Under Construction

This project is very much a work in progress. Expect missing features, placeholder screens, questionable design decisions, and the occasional bug that's somehow been there for three hours.

##  What It'll Do (eventually)

- Track every gadget in the Pocket; status, condition, energy, quantity
- Organize gadgets into categories
- Handle borrowing & returns (looking at you, Nobita)
- Plan missions and pack the right gear
- Generate stats on what's used most, favorites, and more
- Save and load everything locally with persistent storage.

## Built With

C++ · Console UI · Object-Oriented Design · sfml (maybe)

## Why?

A personal learning project to get properly comfortable with Programming Fundamentals + OOP by building one real, cohesive application instead of a pile of disconnected assignments.
